Customer onboarding process

Please follow the step below for processing customers application

Objective:
  • Process Lead conversion to customer and activate service.

Assumptions:
  • The Lead's location is feasible for service activation.

STEPS


SALES HEAD/MANAGER | CDMG-ADMIN

  1. If the Lead came from Sales Rep, Sales Head/Manager will assign the Lead to Agent/Sales rep. If the Lead came from Other source, CDMG-Admin will assign the Lead to Agent/Sales rep.
  2. If the Lead came from Sales Rep, Sales Head/Manager will create Task using Customer onboarding template and assign the task to Agent/Sales rep. If the Lead came from Other source, CDMG-Admin will create Task using Customer onboarding template and assign the task to Agent/Sales rep.

SALES REP

  1. Check Lead information and internet service quotation, mark checklist #1,#2, set Pipeline to: "Qualification". Set Workflow status to: "New"
  2. Get required documents from the Lead, mark checklist #3, set Pipeline to: "Verification". Set Workflow status to: "In-progress - Verification".
  3. Assign the next step to CDMG Group

CDMG-ADMIN

  1. Verify submitted documents and get additional information (ID, Birthday), if no document, assign the task back to Sales Rep and add comment "no documents". go back to Step 3. Mark checklist #4, set Pipeline to: "Verification". Set Workflow status to: "In progress - Verification".
  2. [For unpaid pre-payment]: do a follow up to the Lead via email and/or coordinate with Sales Rep for assistance, if reaffirmation is successful, continue to Step 6, otherwise set the the Pipeline status to: "Lost". Assign the task to Sales Head/Manager and add comment: "unpaid pre-payment". Set Workflow status to: "Done".|Process ends here.
    Convert Lead to Customer. Check Lead Quotes, set Pipeline status to: "Won". Click convert and select service quote, check customer information (How did you find us, contract period, Birthday, ID) and internet services before converting to Customer. Set service status to "Pending". Mark checklist #5, set Workflow status to: "In-progress - pre-payment".
    Note: Set Coversion type to: "Create active customer"
  3. Assign the next step to Billing & Collection Group

BILLING & COLLECTION

  1. Change customer status to "New (Not yet connected)" and check service provisioned, if incomplete, convert back to lead, add comment: "no internet service quote" and assign the task back to CDMG group. Go back to Step 6. Welcome message will be sent automatically after an hour, mark checklist #6, set Workflow to: "In-progress pre-payment".
  2. Create and send proforma invoice to customer, mark checklist #7. Set workflow status to: "In-progress pre-payment".
  3. Check if the new Customer paid the pre-payment within 15 days, if not, convert back to Lead, assign the task back to CDMG, add comment: "For follow up, unpaid pre-payment", set Pipeline status to: Verification, then go back to Step 6. Mark checklist #8. Set Workflow status to: "In-progress - Schedule for activation".
  4. Assign the next step to ISP Group

ISP

  1. Assign and pre-config ONU for the New Customer; and schedule for installation. Mark checklist #9 and set Workflow status to: "In-progress - Schedule for activation".
  2. Install ONU to customer and set start date and End date (calculate based on contract period), mark checklist #10 & #11. Set Workflow status to: "In-progress - Scheduled for Activation".
    Note:
    • Change Customer information - Current status to: "Active",
    • Services - Current status to: "Active",
    • Pending status to: "Paused".
  3. Secure signed Acceptance form and Subscription Agreement/Contract, mark checklist #12. Set Workflow status to: "In-Progress - Billing Configuration".
  4. Assign the next step to Billing & Collection Group

BILLING & COLLECTION

  1. Set billing configuration and recurring invoice, mark checklist #13 and set Workflow status to: "Done".
  2. Assign the next step to SALES HEAD/MANAGER (Project lead)

SALES HEAD/MANAGER

  1. [For Lost Lead]: review and acknowledge the status.
    Review and Archived task, check checklist #1 to #13 and set Workflow status to: "Done".
